## Quick Commands

```bash
# View all issues
gh issue list

# View issues by label
gh issue list --label "good first issue"
gh issue list --label "documentation"
gh issue list --label "enhancement"

# View specific issue
gh issue view 3

# Assign issue
gh issue edit 3 --add-assignee username

# Add to project
gh issue edit 3 --add-project "v-secure Development"

# Close issue
gh issue close 3
```
